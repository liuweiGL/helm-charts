## helm-charts
按照 bitnami 规范维护的自定义 chart

## 使用

1. Add the repository to helm
helm repo add eastcoal https://helm-charts.eastcoal.club

2. Update helm repository listing
helm repo update

3. Install the chart
helm install eastcoal/CHART_NAME
